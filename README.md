# Control de Gastos (Flutter)

App de finanzas personales para registrar ingresos/gastos, ver saldo en tiempo real y manejar presupuestos.

## Features
- ✅ Saldo actual + ingresos/gastos
- ✅ Historial con filtros (mes, categoría, tipo)
- ✅ Presupuestos mensuales + alertas
- ✅ Gráfico por categoría (sin dependencias)
- ✅ Export CSV (copiar / guardar)
- ✅ UX moderna (cards, chips, swipe delete)

## Stack
- Flutter + Dart
- Hive (storage local)
